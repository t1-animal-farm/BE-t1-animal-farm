--------------------- DB 생성 ---------------------------
DB 모델 생성하실 때 저희 주특기 시간에 배운 형태랑 조금 다릅니다.
기능 상 동일한데 깔끔하게 만들려고 한거라서 그냥 똑같이 따라 연결해주시면 될거 같아여.


--------------------- 미들웨어 ----------------------------
미들웨어는 middlewares 폴더에다가 각자 만든거 넣어 주시면 될거 같습니다.


--------------------- DB연결 ----------------------------
각자 DB에 연결해서 사용하시고 합칠때 통합디비로 옮길게요


--------------------- 계층 구조 ---------------------------
3 layered architecture로 하시면 됩니다. 폴더 명은 폴더 만들어 놨으니 저렇게 통일하시면 될거 같습니다.(항해 강의대로 따라함)
